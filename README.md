## Use Project
1. You can clone this project and play with it: https://github.com/JulianDavidG07/Vue-and-TailwindCSS.git

2. Then run:
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
